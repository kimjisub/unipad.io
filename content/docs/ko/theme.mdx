UniPad의 테마 시스템을 사용하면 연주 화면의 외관을 커스터마이징할 수 있습니다.
ZIP 파일로 배포되며, 버튼, 배경, 체인 영역의 이미지와 UI 색상을 변경할 수 있습니다.

## 파일 구조

```
my-theme.zip
├── theme.json          (필수)  테마 메타데이터
├── theme_ic.png        (필수)  테마 아이콘/썸네일
├── playbg.png          배경 이미지
├── btn.png             패드 버튼 (기본 상태)
├── btn_.png            패드 버튼 (눌린 상태)
├── phantom.png         가이드 오버레이
├── phantom_.png        가이드 오버레이 변형 (선택)
├── custom_logo.png     커스텀 로고 (선택)
├── chainled.png        체인 LED 모드 이미지 (모드 A)
├── chain.png           체인 기본 상태 (모드 B)
├── chain_.png          체인 선택 상태 (모드 B)
├── chain__.png         체인 가이드 상태 (모드 B)
└── colors.json         UI 색상 커스터마이징 (선택)
```

## 필수 파일

### theme.json

테마의 이름, 제작자, 버전 정보를 담은 JSON 파일입니다.

```json
{
  "name": "My Theme",
  "author": "Your Name",
  "version": "1.0.0"
}
```

| 필드 | Type | 필수 | 설명 |
|------|------|------|------|
| `name` | string | O | 테마 표시 이름 |
| `author` | string | O | 제작자 이름 |
| `version` | string | | 버전 (기본: `"1.0"`) |

### theme\_ic.png

테마 선택 목록에 표시되는 아이콘입니다. 44dp 크기로 렌더링되며 모서리가 둥글게 처리됩니다.

- 권장 크기: 132×132px (xxhdpi)
- 정사각형 이미지 권장

## 이미지 리소스

모든 이미지 파일은 **PNG 포맷**이어야 합니다.
누락된 리소스는 앱의 기본 테마로 대체됩니다.

### 패드 영역

| 파일 | 설명 | 용도 |
|------|------|------|
| `playbg.png` | 연주 화면 전체 배경 | 패드 그리드 뒤에 표시 |
| `btn.png` | 패드 버튼 기본 상태 | 8×8 그리드의 각 패드 |
| `btn_.png` | 패드 버튼 눌린 상태 | 패드를 터치했을 때 |
| `phantom.png` | 가이드 오버레이 | 자동 연주 가이드 등에 사용 |
| `phantom_.png` | 가이드 오버레이 변형 | 짝수 크기 패드에서 2×2 패턴으로 교차 표시 (선택) |
| `custom_logo.png` | 커스텀 로고 | 연주 화면에 오버레이로 표시 (선택) |

### 체인 영역

체인 영역은 두 가지 모드 중 하나를 선택합니다.

#### 모드 A: LED 모드

`chainled.png` 파일이 존재할 때 활성화됩니다.
패드 버튼(`btn.png`)을 배경으로 사용하고, `chainled.png`를 팬텀 레이어에 배치한 뒤 LED 색상을 오버레이합니다.

| 파일 | 설명 |
|------|------|
| `chainled.png` | 체인 LED 이미지. LED 색상이 오버레이로 적용됨 |

#### 모드 B: Drawable 모드

`chainled.png` 파일이 없을 때 활성화됩니다.
각 상태별로 서로 다른 이미지를 직접 표시합니다.

| 파일 | 설명 |
|------|------|
| `chain.png` | 체인 기본 상태 / LED 채널 |
| `chain_.png` | 체인 선택 상태 |
| `chain__.png` | 자동 연주 가이드 상태 |

## 색상 커스터마이징

### colors.json

UI 요소의 색상을 변경할 수 있습니다. 모든 필드는 선택이며, 지정하지 않은 필드는 기본 색상을 사용합니다.
색상 값은 `#RRGGBB` 또는 `#AARRGGBB` 형식의 HEX 문자열입니다.

```json
{
  "checkbox": "#FF5722",
  "trace_log": "#2196F3",
  "option_window": "#424242",
  "option_window_checkbox": "#FF9800"
}
```

| 필드 | 설명 |
|------|------|
| `checkbox` | 옵션 체크박스 색상 |
| `trace_log` | 트레이스 로그 텍스트 색상 |
| `option_window` | 옵션 패널 배경 색상 |
| `option_window_checkbox` | 옵션 패널 내부 체크박스 색상 |

## 예시

### 최소 구성 (필수 파일만)

```
minimal-theme.zip
├── theme.json
└── theme_ic.png
```

모든 리소스가 앱 기본 테마로 대체됩니다. 테마 목록에 아이콘과 이름만 표시됩니다.

### 전체 구성

```
full-theme.zip
├── theme.json
├── theme_ic.png
├── colors.json
├── playbg.png
├── btn.png
├── btn_.png
├── phantom.png
├── phantom_.png
├── custom_logo.png
└── chainled.png
```

## 설치 방법

1. 위 구조에 맞게 파일을 준비합니다.
2. 모든 파일을 단일 ZIP 파일로 압축합니다. 파일은 ZIP 루트에 위치해야 합니다 (하위 폴더 안에 넣지 마세요).
3. UniPad 앱에서 **Theme → Add Theme → Import ZIP file**을 선택합니다.
4. ZIP 파일을 선택하면 자동으로 검증 및 설치됩니다.

## 유효성 검사

가져오기 시 다음 항목이 검증됩니다.

- `theme.json`이 ZIP 루트에 존재해야 합니다.
- `theme_ic.png`가 ZIP 루트에 존재해야 합니다.
- `theme.json`은 유효한 JSON이어야 하며, `name`과 `author` 필드가 필수입니다.

검증에 실패하면 테마가 설치되지 않고 오류 메시지가 표시됩니다.

<Callout title="디자인 팁">
- 이미지에 투명 배경(알파 채널)을 사용하면 LED 색상이 자연스럽게 블렌딩됩니다.
- `btn.png` 위에 LED 색상이 오버레이되므로, 어두운 톤의 이미지가 효과적입니다.
- `phantom_.png`는 짝수 크기 패드(예: 8×8)에서 `phantom.png`와 2×2 패턴으로 교차됩니다. 시각적 구분을 위해 약간 다른 디자인을 사용하세요.
- 체인 모드 A(LED 모드)는 하나의 이미지에 LED 색상을 동적으로 적용하여 다양한 상태를 표현합니다.
- 체인 모드 B(Drawable 모드)는 각 상태의 이미지를 직접 제어하여 더 자유로운 디자인이 가능합니다.
</Callout>
